I'm Powen Kao

Fascinated by informatics technology. Focus on embedded system development and IoT application. Work as UI developer. Also an amateur photographer.

About Me

Having strong motivation in exploring new technology and gaining hand on experience. Learning C and C++ as my first programming language to work with development board ranging from Raspberry Pi to TI CC2541 and Freescale. To bring the power of embedded system to daily life, I also dive into iOS, android development and develop application that works wirelessly through Wifi or BLE with development boards. Docker containers are used to setup back-end service (Ruby on Rail, nginx...) for these IoT application. What's more, robotic application is also my top interest. I have done two bachelor projects regarding indoor navigation and mobile robotic arm. Both projects are built on ROS and apply Monte Carlo Localization and Inverse Kinematic respectively. More recently, I find game development interesting and engage myself in Unity3D(C#) and pygame(Python) framework to build games. Basic Photoshop and Illustrator skills are acquired while making game asset.
